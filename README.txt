JackpotBandit Setup Guide:

1. Go to https://vercel.com and sign in with your GitHub account.
2. Create a new project and import this folder (upload the contents or push to GitHub).
3. Vercel will auto-detect and deploy the front-end.
4. Use the admin panel at /admin to create players and manage credits.
5. Test the sportsbook at /sportsbook and casino games at /casino.

Default logins:
- Admin: admin@jackpotbandit.com / password: admin123
- Player1: player1@jackpotbandit.com / password: test123
- Player2: player2@jackpotbandit.com / password: test123

To switch to a backend with real odds, add a sports odds API key into the backend .env file.
